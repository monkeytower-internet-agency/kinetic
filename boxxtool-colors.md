# Boxxtool Farbtabellen

Folgende Farben sind für die Boxxtool-Paneele verfügbar (extrahiert aus den Swatches):

| Farbe | RAL / Code | Hex-Code |
| :--- | :--- | :--- |
| **Translucent** | - | `#f5f5f5` |
| **Hellgrau** | RAL 7035 | `#c5c7c4` |
| **Anthrazit** | RAL 7016 | `#383e42` |
| **Klartürkis** | RAL 190 50 45 | `#0b8a81` |
| **Märzgrün** | RAL 100 80 80 | `#add400` |
| **Melonengelb** | RAL 1028 | `#ff9b00` |
| **Verkehrsrot** | RAL 3020 | `#c1121c` |

*Hinweis: Diese Farben können für die UI-Gestaltung (z.B. Kartenrückseiten, Akzente) verwendet werden.*
